# cbe-retail
Extensions to the Common Business Entities framework for Retail Operations. See [Semprini/cbe](https://github.com/Semprini/cbe) for more details.

Extensions include:
  - Sale
  - Product
  - Pricing
 
Entities inherited from CBE:
  - business_interaction
  - customer
  - location
  - party
  - trouble
  - physical_object
  - supplier_partner
  - resource
